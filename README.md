# ReconFlow Commerce MVP — Static

Это автономная статическая версия MVP без npm и build.

## Как опубликовать
1. Создай новый репозиторий на GitHub.
2. Загрузи файлы из этой папки.
3. В Settings → Pages включи GitHub Pages для ветки `main` и папки `/root`.
4. GitHub выдаст публичную ссылку на сайт.

## Альтернатива
Можно просто импортировать папку в любой статический хостинг, который принимает HTML/CSS/JS.
